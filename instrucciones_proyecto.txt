INSTRUCCIONES PARA EJECUTAR EL PROYECTO NODE.JS

1.  REQUISITOS

-   Tener instalado Node.js (versión LTS recomendada)
-   Tener npm (incluido con Node.js)

Verificar instalación: node -v npm -v


ACCEDER AL PROYECTO Abrir una terminal en la carpeta del proyecto:
    cd aplicacionnode

INSTALAR DEPENDENCIAS Ejecutar: npm install

EJECUTAR LA APLICACIÓN Ejecutar: node app.js

O en algunos casos: npm start

ACCEDER DESDE EL NAVEGADOR Abrir: http://localhost:3000

(Si no funciona, revisar el puerto en el archivo app.js)

 DESCRIPCIÓN DEL PROYECTO Aplicación web desarrollada con Node.js y
    Express que utiliza EJS como motor de vistas y archivos JSON como
    almacenamiento de datos.

POSIBLES ERRORES Y SOLUCIONES

-   Error: Cannot find module Solución: ejecutar npm install

-   Error: puerto ocupado Solución: cambiar el puerto en app.js

-   Error: no carga JSON Solución: verificar rutas de los archivos .json
